# Glossary

| Term | Definition |
|------|------------|
| **Enquiry** | A sales enquiry or project, identified by an external system ID |
| **External Enquiry ID** | Unique identifier from external CRM/ERP system |
| **CustomerSku** | Customer's specific product with dimensions and weight |
| **CustomerPallet** | Customer's pallet specification with capacity |
| **WarehouseConfig** | Building parameters (dimensions, height, constraints) |
| **RackConfiguration** | The racking system design for an enquiry |
| **Snapshot** | Named point-in-time copy of configuration state |
| **Version** | Sequential number tracking configuration changes |
| **Configuration State** | JSON representation of complete configuration data |
| **Status** | Lifecycle state: Draft, InProgress, Submitted, Converted, Archived |
