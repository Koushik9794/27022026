# Ubiquitous Language

Dictionary of terms used in the Catalog Service domain.

| Term | Definition | Context |
|------|------------|---------|
| **SKU** | Stock Keeping Unit | Unique identifier for a catalog item |
| **Code** | Human-readable identifier | `SKU-001`, `PAL-EURO`, `MHE-RT-01` |
| **GLB** | 3D model binary file | Attached to SKUs for visualization |
| **Active** | Component is available for use | Can be included in configurations |
| **Deleted** | Soft-deleted state | Not shown in queries, recoverable |
| **Pallet Type** | Classification of pallet | `EURO`, `US`, `CUSTOM` |
| **Equipment Type** | Classification of MHE | `FORKLIFT`, `REACH_TRUCK`, `ORDER_PICKER` |
| **Palette** | UI menu configuration | Component groups for DesignDeck |
| **Palette Group** | Category in UI menu | `Warehouse Types`, `SKUs`, `MHE` |
| **Element** | Individual palette item | Renderable component in configurator |
