# Glossary

| Term | Definition |
|------|------------|
| **SKU Type** | Type of customer product/goods to be stored (Box, Drum, Bin, Gunny Bag). Defines the attribute schema for that type. |
| **Pallet** | A load platform for stacking goods in warehouse storage |
| **MHE** | Material Handling Equipment — forklifts, reach trucks, VNA trucks, etc. |
| **ComponentType** | Type of racking component (Beam, Upright, Base Plate) from the taxonomy |
| **ProductGroup** | Family of racking products (SPR, VNA, Cantilever) |
| **Attribute Schema** | JSON definition of what attributes a SKU type or component requires |
| **Palette** | UI configuration for the DesignDeck component menu |
| **Aggregate** | DDD pattern — a cluster of objects treated as a unit |
| **Soft Delete** | Marking a record as deleted without physical removal |
| **GLB File** | 3D model file format (Binary glTF) |
| **CQRS** | Command Query Responsibility Segregation pattern |
| **WolverineFx** | .NET messaging framework used for CQRS |
| **Dapper** | Micro-ORM for database access |
| **FluentMigrator** | Code-based database migration framework |
