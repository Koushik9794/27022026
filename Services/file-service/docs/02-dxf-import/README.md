# DXF Import & Civil Layout

## Purpose
To parse DXF files uploaded by users, specifically looking for Civil layout information that informs site configuration.

## Key Features
- **Layer Extraction:** Ability to extract specific layers relevant to site boundaries or obstructions.
- **Entity Parsing:** Identify and parse entities like polylines, circles, and text.
- **Coordinate Transformation:** Convert DXF coordinates to internal system coordinates if necessary.
- **Metadata Extraction:** Extract available metadata from the DXF header.
